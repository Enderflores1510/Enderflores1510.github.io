Algoritmo del punto medio para circunferencias

Definición:

El algoritmo del punto medio para circunferencias es un algoritmo usado para determinar los puntos necesarios para rasterizar una circunferencia. El algoritmo se puede generalizar a curvas cónicas.

Funcionamiento

Similar al algoritmo de la circunferencia de Bresenham, se parte de un círculo con centro (xc, yc) y radio r. Se inicializa un valor d que determina el comportamiento de la curva.

Dependiendo del valor de d, se decide si el siguiente punto debe ser a la derecha o hacia abajo y hacia la derecha (en términos de coordenadas). Al igual que el algoritmo anterior, los puntos calculados se reflejan sobre los ocho octantes de la circunferencia. 

                        
                        void CircunferenciaPuntoMedio(int xc, int yc, int r) {
                            int x = 0, y = r;
                            int d = 1 - r;
                    
                            setcolor(GREEN);
                    
                            while (x <= y) {
                                putpixel(xc + x, yc + y, GREEN);
                                putpixel(xc - x, yc + y, GREEN);
                                putpixel(xc + x, yc - y, GREEN);
                                putpixel(xc - x, yc - y, GREEN);
                                putpixel(xc + y, yc + x, GREEN);
                                putpixel(xc - y, yc + x, GREEN);
                                putpixel(xc + y, yc - x, GREEN);
                                putpixel(xc - y, yc - x, GREEN);
                    
                                if (d < 0) {
                                    d = d + 2 * x + 3;
                                } else {
                                    d = d + 2 * (x - y) + 5;
                                    y--;
                                }
                                x++;
                            }
                        }

Entrada

Centro de la circunferencia (xc, yc) y radio r.

Inicio

Se inicializan las variables x = 0 y y = r, que representan las coordenadas del punto en el perímetro de la circunferencia.

Cálculo del error

La variable d es usada para controlar el proceso de dibujar la circunferencia. Se inicia con d = 1 - r, que es el error inicial.

Bucle principal

A medida que se dibujan los puntos de la circunferencia, la función va actualizando x y y y calcula el nuevo valor de d en cada iteración.

Si el error d es negativo, se ajusta sumando a d el término 2 * x + 3, moviendo el punto hacia la derecha.

Si el error es positivo o igual a cero, se ajusta con 2 * (x - y) + 5, moviendo el punto en diagonal hacia abajo y hacia la derecha.

Simetría

Como una circunferencia es simétrica, se dibujan ocho puntos en cada iteración, aprovechando la simetría con respecto a los ejes y las diagonales.

Condición de salida

El bucle termina cuando x supera a y, lo que indica que la circunferencia ha sido trazada por completo.
